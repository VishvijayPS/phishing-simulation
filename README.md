# Phishing Simulation

A simple GUI-based website that simulates various phishing attack scenarios for educational and security awareness purposes.

## Overview

This project demonstrates different types of phishing websites in a graphical user interface format. It is designed to help users understand and recognize phishing attempts and improve their security awareness.

## Project Structure

```
phishing-simulation/
├── client/           # Frontend client application
├── server/           # Backend server
├── files/            # Project files and resources
├── logs/             # Application logs
├── package.json      # Node.js dependencies
├── package-lock.json # Locked dependency versions
└── README.md         # This file
```

## Features

- **GUI Interface**: User-friendly graphical interface for phishing simulation
- **Multiple Scenarios**: Various phishing website examples
- **Educational Tool**: Perfect for security awareness training
- **HTML-based**: Built with HTML for easy deployment

## Getting Started

### Prerequisites

- Node.js and npm installed on your system

### Installation

1. Clone the repository:
```bash
git clone https://github.com/VishvijayPS/phishing-simulation.git
cd phishing-simulation
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

To start the application:
```bash
npm start
```

This will launch the server and you can access the application through your browser.

## Project Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js
- **Runtime**: Node.js with npm

## Directory Structure

- **client/**: Contains the frontend code for the GUI interface
